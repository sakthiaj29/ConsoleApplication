

**<u>Library Management System</u>**

Target audience – Admins, User

Development time – 3 Days

**<u>Admin :</u>**

- Admin login - P1

- Set library on first time – P1

- Add books – P1

- Add users - P1

- View list of books - P1

- View search books – P1

**<u>Candidate :</u>**

- Candidate login - P1

- View list of books - P1

- View search books – P1

- Book borrow – P1

- Return book – P1

- View list of borrowed books – P1
